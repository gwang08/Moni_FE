# Moni - IELTS Practice App

A modern IELTS practice web application built with Next.js 15, Tailwind CSS, and shadcn/ui.

## Features
- Landing page with dashboard preview
- Target score tracking with localStorage
- Exam countdown timer
- Activity calendar heatmap
- 4 IELTS skills: Reading, Listening, Writing, Speaking

## Tech Stack
- Next.js 15 (App Router)
- Tailwind CSS v4
- shadcn/ui
- Zustand (state management)
- TypeScript

## Getting Started
```bash
npm install
npm run dev
```

Open http://localhost:3000
